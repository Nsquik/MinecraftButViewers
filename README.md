# MinecraftButViewers

## Website for MINECRAFT streamers that lets viewers respawn creepers and other mobs, apply an effect on a streamer, play sounds on him for money.

# Stack

## Front-End

- React Context API

- React Hooks API

- React Router

- Axios

- Socket.io

## Back-end

- Node.js

- MongoDB ( mongoose )

- Express

- Passport

- Cookie-session

- Paypal SDK

# How to deploy on heroku

1. Clone this repo.
2. Install heroku CLI
3. Create Heroku App

```
    heroku create
    git remote -v // Confirm that a remote named heroku has been set.
```

4. Deploy code

```
git push heroku master
```
